    NEOS-X

    Busque por asteroides próximos da terra
    Ideal para aficcionados da Astronomia

    Fácil, um único script PHP

    Através dos APIs oficiais da NASA fornecidos publicamente, podemos agora buscar por asteroides próximos a terra, ver seus detalhes e suas propriedades, este script proporciona acesso rápido a esta informação, você pode usar no seu projeto de astronomia para rastrear estes objetos com seu telescópio ou usar estes dados da forma que deseje. 
    Para um melhor entendimento veja a documentação toda no GITHUB.

    Ver a Demo Online (Full Version)

    Open Source! 

    Enquanto a NASA permitir o uso da API de forma gratuita, prometemos que este código continua sendo de uso público e Open Source, faça o uso que melhor se adaptar a suas necessidades. 

    Nós temos 2 versões do script uma que exibe informações limitadas para ser visualizado em um celular ou tablet, e outra versão que pode ser usada no desktop com muito mais informação disponível na tela.
